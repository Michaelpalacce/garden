Everything we work on has **activation energy**. It takes time to get in the [[Flow]]. 

> It takes time to pick up what you paused briefly. The same way a car requires the most power when actually starting.

# Single Task Focus

If we focus on a single task for longer periods of time, it will yield more results. 

- How to do this?
	- You can create [[Time Boxing|Focus Blocks]] in your calendar for this. 
	- You can utilize the [[Pomodoro Technique]]
